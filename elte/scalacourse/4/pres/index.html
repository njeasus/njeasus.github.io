Tipuskorlatok

A Scala tipusrendszereben az altipusossag (<:) reszbenrendezes

Reflexiv: A <: A

Tranzitiv: A <: B es B <: C => A <: C

Antiszimmetrikus: A <: B es B <: A => A =:= B

Variancia

Annak a meghatarozasa, hogy egy tipuskomponens altipusossaga hogyan befolyasolja a befoglalo tipus altipusossagat

Befoglalo tipus: generikus, komponens: tipusparameter

Liskov-fele helyettesithetosegi elv

Nincs koztuk relacio: invariancia

Komponens altipusossaga oroklodik: kovariancia

A <:B => Generic[+A] <: Generic[B]

Komponens altipusossaga megfordul: kontravariancia

A <:B => Generic[-A] >: Generic[B]

Kovariancia

class List[+A]
				
trait Animal { def name: String }
case class Cat(name: String) extends Animal
case class Dog(name: String) extends Animal

def printAnimalNames(animals: List[Animal]): Unit = {
  animals.foreach { animal => println(animal.name) 
}
printAnimalNames(List(Cat("Whiskers"), Cat("Tom")))
printAnimalNames(List(Dog("Fido"), Dog("Rex")))

Bemeno parameter: nem lepi meg a belso kontextet

Kontravariancia

trait Printer[-A] { def print(value: A): Unit }
class AnimalPrinter extends Printer[Animal] { 
  def print(animal: Animal): Unit = println("The animal's name is: " + animal.name) }

class CatPrinter extends Printer[Cat] { 
  def print(cat: Cat): Unit = println("The cat's name is: " + cat.name) }

val myCat: Cat = Cat("Boots")
def printMyCat(printer: Printer[Cat]): Unit = {
    printer.print(myCat)
}

printMyCat(new CatPrinter)
printMyCat(new AnimalPrinter)

Variancia es also korlat

case class Box[+A](value: A) { def sit(a: A): Box[A] = Box(a) }

covariant type A occurs in contravariant position in type A of value a

val catBox = new Box[Cat]
val animalBox: Box[Animal] = catBox // Cat <: Animal
val dog = new Dog
animalBox.sit(dog) //Uh-oh kutyat ultettunk a macskadobozba

Family polymoprhism

Tipusmezo => path dependent

trait Family {
    type M <: Mother; type F <: Father

    class Father(val name: String) { def kiss (m:M) = ??? }
    class Mother(val name: String)
}

object UpperClassFamily extends Family {
    type F = Father; type M = Mother

    class Mother (name: String , val lastName : String ) extends super.Mother (name)
}

object MiddleClassFamily extends Family {
    type F = Father; type M = Mother
}

val jonesFather = new MiddleClassFamily.Father("John")
val smithsMom = new UpperClassFamily.Mother ( "Kaitlyn","Smiths" )
jonesFather.kiss(smithsMom) //Uh-oh UpperClassFamily.M != MiddleClassFamily.M
