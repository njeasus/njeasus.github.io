Osztalyok

class C { 
    private val x: Int = 3
    def printX() = println(x)
}
val myObject = new C
myObject.printX()
abstract class AC { 
    protected def x: Int = 3
    def printX() = println(x)
}
val myObject = new AC // error: class AC is abstract; cannot be instantiated
class IC extends AC { 
    override val x: Int = 4
}
val myObject = new IC
myObject.printX() // 4
class IC extends AC with C { }
// error: class C needs to be a trait to be mixed in

Linearizacio

Osztaly -> extends linearizalt hierarchia -> with linearizalt hierarchia

Minden egyeb field es metodus is elerheto

extends ... with ... with ...

Minden egyeb field es metodus is elerheto: super[T]

trait A { def x = 3 }
trait B { def x = 3 }
class AB extends A with B //error: class AB inherits conflicting members
class AB extends A with B {
    override def x = super[A].x
}

class BA extends B with A {
    override def x = super[B].x
}
